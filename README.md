# 🎮 FPS BATTLE — Online Multiplayer FPS

## 起動方法 (How to Run)

### 1. 依存パッケージをインストール
```bash
npm install
```

### 2. サーバーを起動
```bash
node server.js
```

### 3. ブラウザでアクセス
```
http://localhost:3000
```

---

## 🔑 管理者パスコード
```
java
```

---

## 🎯 ゲームの流れ

1. **タイトル画面** → 「MULTIPLAYER」をクリック
2. **確認画面** → 「YES」を選択
3. **サーバー選択** → 事前に管理者パネルでサーバーを作成してから参加
4. **ゲーム開始** → クリックでPointerLockを取得

---

## ⚙️ 管理者パネル (Admin Panel)

画面右上の「⚙ ADMIN」ボタンから開きます。

| 機能 | 説明 |
|------|------|
| サーバー作成 | 名前・最大人数を設定して作成 |
| サーバー削除 | 参加者を全員キックして削除 |
| 最大人数変更 | 各サーバーの上限を変更 |
| プレイヤーキック | 個別プレイヤーを切断 |

## 🕹️ 操作方法 (Controls)

| キー | アクション |
|------|-----------|
| WASD / 矢印キー | 移動 |
| マウス | 視点移動 |
| 左クリック | 射撃 |
| 右クリック | エイム (ADS) |
| 1 | アサルトライフル |
| 2 | ショットガン |
| 3 | スナイパーライフル |
| Space | ジャンプ |
| Tab | スコアボード |
| T | チャット |
| ESC | PointerLock解除 |

## 🔫 武器仕様 (Weapons)

| 武器 | ダメージ | 弾数 | 連射 |
|------|---------|------|------|
| アサルトライフル | 25/発 | 30発 | 自動 |
| ショットガン | 15×8散弾 | 8発 | 単発 |
| スナイパー | 90/発 | 5発 | 単発+ボルト |

## 🗺️ マップ概要

- **中央**: 時計塔 — 両チームの激戦ポイント
- **左奥**: 赤チームリスポーン基地 (赤いゲート)
- **右奥**: 青チームリスポーン基地 (青いゲート)
- **中央左右**: 赤いコンテナ — 遮蔽物
- **全体**: コンクリートバリア、木箱、サンドバッグ

### 3レーン構造
- **センター**: 時計塔を挟んだメインルート
- **左レーン**: 左側の建物沿いルート
- **右レーン**: 右側の建物沿いルート

---

## 🔧 システム要件

- Node.js 16+
- モダンブラウザ (Chrome/Firefox/Edge推奨)
- PointerLock API対応

---

## ⚡ 技術スタック

- **サーバー**: Node.js + Express + Socket.io 4
- **クライアント**: Three.js r160 (WebGL)
- **通信**: WebSocket (Socket.io)
- **3D**: Three.js (PerspectiveCamera, Raycaster, MeshLambertMaterial, etc.)
